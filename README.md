# 📊 Data Jobs Dashboard (Power BI)

## 📌 Overview
An interactive Power BI dashboard analyzing global data job postings.  
It highlights job demand, salary trends, roles, and hiring platforms.

---

## 🚀 Key Insights

- 479K total job postings
- $113K median yearly salary
- $47.62 median hourly salary
- Senior roles offer the highest pay
- LinkedIn is the top hiring platform
- Shift toward skill-based hiring

---

## 🔍 Features

- Job trends over time (2024)
- Salary comparison by role
- Highest paying data jobs
- Drill-through analysis (e.g., Senior Data Scientist)
- Global job distribution map
- Hiring platform comparison

---

## 🛠️ Tools

- Power BI
- Power Query
- Dataset: `job_postings_flat`
